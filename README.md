# 📖 Warm Stories

> A warm, cozy, editorial-style blog with a storytelling focus.

![Theme](https://img.shields.io/badge/Theme-Warm%20%26%20Cozy-brightgreen)
![Style](https://img.shields.io/badge/Style-Editorial-orange)
![GitHub Pages](https://img.shields.io/badge/Hosting-GitHub%20Pages-brightgreen)

## 🌐 Live Demo

**https://khanblair.github.io/warm-stories/**

## ✨ Features

- 🏠 **Homepage** - Featured stories, categories, newsletter signup
- 📝 **Blog Page** - Article listings with thumbnails and excerpts
- 👤 **About Page** - Author story and mission
- 📧 **Contact Page** - Get in touch form
- 🎨 **Warm Color Palette** - Cozy cream, terracotta, and sage tones
- 📱 **Fully Responsive** - Mobile and desktop optimized
- 🍂 **Seasonal Sections** - Curated content by season

## 📁 Pages

| Page | Description |
|------|-------------|
| [index.html](index.html) | Homepage with featured stories, categories, newsletter |
| [blog.html](blog.html) | All blog posts with thumbnails and excerpts |
| [about.html](about.html) | About the blog and author |
| [contact.html](contact.html) | Contact form and social links |

## 🛠️ Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, Flexbox, Grid
- **Vanilla JavaScript** - No frameworks
- **Google Fonts** - Playfair Display & Lora

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/khanblair/warm-stories.git

# Navigate to project
cd warm-stories

# Open directly in browser or use a local server
python3 -m http.server 8000
```
